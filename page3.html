1.7 Describe which are the appropriate filetypes for websites

There are many file types that are supported by HTML but the main ones that make a webpage run are:
HTML: Hyper Text Markup Language is the framework for any webpage while looking blank without the other files it needs in order for CSS and Javascript to be utilized fully.
Javascript: Javascript allows websites to run more than just simple processes allowing the website to make automated processes when a select part of the website is used.
CSS: Cascading Style Sheets is mainly used for allowing the website to look the way a creator wants it to be such as changing fonts, color and size
